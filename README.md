# README

Forum is a sample project that allows users to ask and answer the questions
### Requirements

- Ruby >= 3.2.1
- Rails >= 7.0.7.2

### Setup
- clone the this repo
- cd into the cloned repo, run `bundle install`
- rails db:create
- run `rails server` to start the server
